Project Overview

This E-Portfolio website represents my current skills and learning journey in the Software Engineering program at Coding Temple. Built using Bootstrap framework, it demonstrates my ability to create professional, responsive web applications using modern web technologies.

Key Pages & Features

Homepage

Features hero section, project cards with images, and interactive tables showcasing upcoming projects.

About Page

Personal background, skills, and goals presented in an engaging modal format.

Projects Page

Comprehensive showcase of completed and ongoing projects with detailed descriptions.

Contact Page

Professional contact forms and social media links for networking opportunities.

Challenges & Solutions

Challenge: Design Consistency

Maintaining consistent styling and user experience across multiple pages.

Solution: Centralized CSS

Created a unified styles.css file with shared components and consistent color scheme.

Challenge: Background Integration

Making text readable over complex background images while maintaining design appeal.

Solution: Fixed Background + Overlays

Implemented fixed background with dark overlays and text shadows for optimal readability.
